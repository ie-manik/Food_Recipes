# Cooking Recipes Website

This is my web development assignment for 2nd semester. I made a cooking recipes website using only HTML and CSS (no javascript).

---

## About

Its a multi page website that shows different recipes. I tried to make it look clean and simple. The website has a home page and 5 recipe pages.

---

## Pages

- Home page
- Creamy Pasta Alfredo
- Butter Chicken
- Paneer Tikka
- Chocolate Lava Cake
- Chicken Biryani

---

## What I Used

- HTML
- CSS (flexbox, grid, media queries)
- Google Fonts
- No javascript at all

---
